遊戲介紹

以下的教程及資料可以幫助你在工業系統中了解新的目標，你也可以做更多的事情。 如果你是一名不知道在遊戲中該做什麼以及怎麼做的新手，你可以參考一下本頁面中的文章以及在 Discord 詢問他人。

　1-----開始

　　開始 基本上前期遊戲就跟原始Minecraft沒有兩樣，也是收集材料並且進入石器時代
但... 在工業世界裡，沒有直接的鐵礦擺在你眼前，取而代之的是包含著各種礦石的礦砂，他需要被提煉才能成為你要的物質。
 

　2-----挖礦

　　在工業世界裡有幾種礦產，分別是雜礦砂、貴礦砂以及精礦砂(分別取代原版的 鐵礦、紅石礦以及金礦，所以你可以用原版找鐵找金的位置去找工業的特殊礦物喔！)
雜礦砂，一種最基本的礦石，你可能會疑問要如何煉取礦物?
首先你必須先合成礦物煉製台，當然先從最基礎的煉製台開始
首先先將四根木棍、三條線丟在"合成台"上 便會獲得到一個煉製台，點右鍵便能放出
放出後你會發現出現一個煉製台模型，煉製台下方會有個鵝卵石(敲掉鵝卵石便能破壞煉製台)鵝卵石上方工作區域，將礦砂"丟"在上頭便能開始煉製(當然最基本煉製台只能煉製雜礦砂，其他高階礦物需要更高階的融煉台)
煉製後每個雜礦砂會隨機變成一個礦物(此時物資應該能進入鐵器時代了)

　3-----合成

　　工業系統的特殊物品很多事不能用普通合成台直接合成的，你需要更高階的工業合成台才能合成
將 一個普通合成台 一個鐵磚 一個鐵斧 一個鐵鍬 一個鐵稿 丟在普通合成台上便能獲得工業合成台
點右鍵可以放置於地上，你會看到放置後會出現一個投擲器和工業合成台，敲掉投擲器便能破壞工業合成台
對投擲器點擊右鍵可開啟合成介面，接著各種道具的合成配方可能得參閱位於本群組的資料區的其他文章了！
 

　4-----研究

　　你可能在嘗試合成一些東西時發現明明東西沒放錯卻無法合成，那是因為你的科技不足
你需要進行研究，首先先合成出一個工業研究桌(合成需要用工業合成台，配方詳見 #科技 最上方)
合成後你會獲得到研究桌，右鍵可以進行放置，放置後你會看到一個黑色羊毛及研究台，站在黑色羊毛上按下shift便會出現研究列表在聊天欄 他會顯示你現在能研究的項目
研究項目需要消耗物資，將她所需的物資放在副手再點擊文字訊息的研究便能完成研究
有些科技需要更高階的研究台才能研究！(如果是多人模式的情況之下，一個人研究一項科技，所有人都能獲得該科技)
 

　5-----電力

　　電能，自然界中一種強大的動能，當然在這個系統中更是佔有一席之地啊！在知道如何研究及合成後讓我們來進行電子設備的製造讓文明進入新的領域吧！
電力不能少的就是發電機了，讓我們先從最基本的光能發電板開始吧！(合成配方詳見 #電力 和 #零件 )
首先先製造出重要零件"基礎電子零件"後再製造出光能電版，可以按右鍵放置，請放置在能照到自然光的地方，放置後你會看到一個日光感測器，對其點擊右鍵可更換日光或月光模式(可以改變發電板是要在白天還是要再晚上發電，破壞日光感測器能破壞光能電版)再製造出光能電版後接下來最重要的就是將電能供給給我們的電器
先從最基本的電爐開始吧！(合成配方詳見 #電力 和 #零件 )
不論是任何的發電設備所發出的電剛開始都是高壓電能，需要使用高壓電塔進行傳送(合成配方詳見 #電力 和 #零件 )高壓電塔放置在發電廠附近64格內幾可接收電能，如果電器距離很遙遠可以再興建第二座以上的電塔放在於第一座電塔附近64格內便能互相傳遞電能了，然而我們的電器是不吃高壓電的，須將其轉為常壓電，在高壓電塔附近興蓋變壓器(合成配方詳見 #電力 和 #零件 )此時我們可以再興建常壓電塔在變壓器附近，如果電器距離遙遠可以再用好幾座常壓電塔互相連結來傳輸電能，只要電器在常壓電塔附近便能獲得電能了！一個簡單的電網便興建完成了(不過要特別注意喔！使用常壓電流長距離輸電會浪費大量電能)。
 

　6-----經濟

　　在剛剛我們建立出電爐，電爐可以透過消耗電力來產生熱能來，不需要煤炭或岩漿也能融煉物資，我們還能夠來建立石油煉製機來提煉塑膠製造更多工業產品
或者打造洲際軌道傳送艙來快速移動上千格，當然如果你厭倦挖礦的生活你可以建立挖礦機消耗電能來挖礦甚至植樹機及農業機都能讓你更輕鬆的遊戲(詳見 #民生 )
 

　7-----軍事

　　當我們的經濟工業已經到了新的領域，沒想到躲在樹林里的那些噁心生物卻發出sss的聲音把一切都炸了，是時候開發軍事武器來防衛自我了
這些機槍炮可以全自動攻擊所有在附近的敵人(詳見 #軍事 )
更好的裝備及武器讓一切變得更容易生存變得更簡單。

　8-----工具

　　此時你可以製造一些工具，它可以更快速的改造你的家園(詳見 #工具 )
 

　9-----太空

　　在一切都準備就緒後，是時候突破地心引力的束縛，前往下個星球探索了！
建立起火箭發射太空梭向各個星球前進！當然星際工業是非常燒錢的，不過每個星球都會有它特殊的物資特產，也許你能獲得到不錯的探索喔！(在當前版本並沒有氧氣問題，也不需要穿戴太空裝，不過在未來考慮更新) 詳見 #星際
 

　10-----目標

　　突然間，我們星艦來到一個廢棄的太空站，這曾經是人類的殖民星艦，可惜上頭只留下許多完全失控的防禦機器人，以及一個殺人機器，也許能摧毀這些機器人讓曙光重回人類之上！
詳見 #星際

如何安裝

安裝載點及須知

⋘ 工業革命。重啟 Industrial Revolution ⋙

Minecraft版本： 1.17+

安裝方式：

1. 創建新世界。
2. 開啟遊戲資料夾，進入剛剛創建的世界的資料夾，並將下載後的檔案解壓縮，將其/industry資料夾放入/datapacks資料夾中。
3. 回到遊戲，打/reload指令並稍等數秒。
4. 他會跳出聲明書！輸入下載材質包時提供在檔案旁邊的密碼以激活系統
5. 輸入四碼正確密碼後安裝完成。

更新方式(如果地圖已經安裝過本系統要更新版本方式)：

如果已安裝過舊版生存系統僅需要刪除原本舊的/industry資料
並將重新下載下來的壓縮檔中的/industry資料夾放入/datapacks資料夾中
回到世界中輸入/reload指令，並重新輸入四位密碼僅可
如果沒有安裝過請參考上面的安裝方式，謝謝

載點：

    指令包 材質包

請務必安裝材質包，否則無法激活系統！
