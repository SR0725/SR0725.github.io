魔法空島

前言

空島生存，從Minecraft出現以來就是很常被使用的題材，什麼時候我們才能脫離單純的疊方塊呢？
而魔法元素也常是被加入在各大遊戲中的素材，如果我們將二者合一
是否能擦出新的火花呢？
抱持著實驗精神，從沒開發過空島生存地圖的睿拾起了他的設計書，將夢想實踐為現實！

  • Bug回報

作者的話

這張地圖的開發從企劃到完成大約僅使用上一個禮拜的時間
中間又經歷過1.17的更新
如果在遊戲中有遇到任何錯誤，歡迎回報
這也是本人第一個開發的空島地圖
希望各位能開心遊玩

劇情

宇宙混沌過後，世界出現幾顆生命之種
他們發芽，結果，最後創造了世界的空島
人類的文明在這裡成長，茁壯，然而...
大約在上百年前，邪魔進入這個世界
世界受到毀滅性的打擊，只留下了六大島嶼
幸好勇者挺身而出，犧牲自己封印了魔王
然而近年來魔王卻蠢蠢欲動...

遊戲設定

顯示設定

  1. 亮度：明亮
  2. 雲層：關閉
  3. 視野距離：8以上
  4. 粒子特效：少量以上

音樂和音效

  1. 主音量：不為0%
  2. 音樂：關閉
  3. 玩家：不為0%（此為遊戲音效）
  4. 其他：無指定

注意事項

  1. 請務必安裝「指定材質包」
  2. 歡迎多人共同挑戰
  3. 版本限定「1.17」
  4. 地圖預設：防爆、防噴
  5. 強制使用Unicode字型：關閉
  6. 電腦性能不佳者，建議至遊戲目錄：選項>顯示設定>動畫>粒子密度
  7. 祝你遊玩愉快
